# ADR 007: MIT license

- **Status:** accepted
- **Date:** 2026-07-17

## Context

The repo needs a license. It is documentation and light tooling, meant for the
widest possible adoption as a portfolio piece.

## Options considered

- **MIT**: permissive, standard for tooling, maximum adoption.
- **CC BY**: natural for docs, but unusual for a repo that also ships code.
- **GPL**: copyleft; discourages the casual reuse we want to encourage.

## Decision

MIT for the whole repo.

## Consequences

- Anyone can use, adapt, and even commercialize it. Accepted.
- The underlying LLM Wiki pattern is credited to Andrej Karpathy in the README
  regardless of license (attribution is an ethical, not licensing, requirement).
